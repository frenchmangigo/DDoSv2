## Usage:

1. Clone the repository:

    ```bash
    git clone https://github.com/joshuapostol/DDoSv2/
    ```

2. Install aiohttp:

    ```bash
    python3 -m pip install aiohttp
    ```

3. Navigate to the directory:

    ```bash
    cd DDoSv2
    ```

4. Run the script:

    ```bash
    python Dosv2.py
    ```

5. Enter the URL of the target website when prompted.
